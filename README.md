# EnglishAI - AI 英语私教应用

一款基于AI的个性化英语学习应用，采用卡片式交互，让学习更高效有趣。

## ✨ 特性

### 🎨 双风格设计
- **主界面**：炫酷的 Persona 风格（霓虹边框、渐变卡片、赛博朋克风格）
- **学习页**：极简清新风格（专注学习内容，减少干扰）

### 🎴 卡片式学习
- 扑克牌质感的单词卡片（3D阴影、厚重边框）
- 流畅的翻转和滑动动画
- 键盘快捷操作：
  - `Z` 键 - 不会，翻转查看释义
  - `X` 键 - 会了，下一张

### 📱 一屏展示设计
- 所有信息在视口内完整展示
- 无需滚动，专注学习
- 100vh 视口适配

### ⚡ 技术栈
- React 18
- Vite
- Framer Motion (流畅动画)
- CSS Variables (主题系统)

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 🎮 使用说明

1. 打开应用，查看炫酷的主界面
2. 点击"DAILY QUEST"开始学习
3. 使用键盘快捷键高效学习：
   - `Z` - 不认识这个单词，翻转查看
   - `X` - 认识/学会了，下一个
